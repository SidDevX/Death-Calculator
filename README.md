# Death Calculator

A dark horror-themed fictional "death date" predictor that generates an eerie glimpse into your supposed final moments.

Enter your personal details and let the algorithm craft a chilling, purely fictional prediction about your fate. Designed with a minimal yet immersive aesthetic, this project uses only HTML, CSS, and JavaScript — no frameworks, no dependencies.

⚠️ Disclaimer: This is purely for entertainment purposes. The predictions are not real and should not be taken seriously.

Features:
- Simple and interactive input system
- Atmospheric horror-themed UI
- Randomized fictional predictions
- Lightweight (no external libraries)
